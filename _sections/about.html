EpiCode is a QRCode alike : It is a bi-dimensional matrix, machine readable, that contains information about the item to which it is attached. The project aim is to develop a software capable of both generating EpiCodes and reading them. Generating an EpiCode will be the processing data part. The input data can be a web address, some text, numbers or any other information, and then it is morphed into an image. On the other hand, we have the reading part, which will involve image processing, as well as decrypting of the information contained in the code.

Moreover the project has some algorithmic challenges considering it involves multiple processes :

  • Data Processing : The EpiCode, like the QRCode, processes data from an image, this process is the data processing, it matches data with a pattern (in the form of a matrix composed of 0 and 1) and then produces an image (aka the EpiCode) out of it.
  • Image Processing: To retrieve the data, the software extracts it from the EpiCode. During this process, the image is modified if needed in order to improve its quality, and overall its clearness.
